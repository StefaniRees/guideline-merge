## Descrição
Branch criada a partir da *main* (ou homologação) para implementar uma mudança específica de forma isolada e rastreável.

## Boas Práticas
- Manter branches **curtas** e **frequentemente integradas** com a branch principal.  
- Adotar **convenção de nomes** clara (ex.: `feature/`, `bugfix/`, `hotfix/`), preferencialmente com ID do ticket/squad.

## Justificativa
Reduz riscos de conflitos tardios, melhora rastreabilidade e facilita auditoria.

## Obrigatório?
**Sim**
